# Xassemblian Weather Model (XWM) 

Antalya'nın merkez 5 ilçesini (Muratpaşa, Konyaaltı, Kepez, Aksu, Döşemealtı) kapsayan, **0.9 km ultra-yüksek çözünürlüklü** mikro-klima tahmin modeli.

##  Teknik Detaylar
- **Model:** WRF (Weather Research and Forecasting) V4.7.1
- **Çözünürlük:** Domain 1 (4.5 km), Domain 2 (0.9 km)
- **Donanım:** AMD Ryzen 7 9700X (8 Cores, 16 Threads)
- **İşletim Sistemi:** Debian Linux (WSL2)
- **Vizyon:** Mahalle bazlı sıcaklık ve rüzgar analizi (Örn: Kızıltoprak Mahallesi).

*Model şu an aktif olarak simülasyona devam etmektedir.*
