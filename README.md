# Xassemblian Weather Model (XWM) 

Antalya'nın merkez 5 ilçesini kapsayan, 0.9 km çözünürlüklü mikro-klima tahmin modeli.

##  Teknik Detaylar
- **Model:** WRF (Weather Research and Forecasting) V4.7.1
- **Çözünürlük:** Domain 1 (4.5 km), Domain 2 (0.9 km)
- **Donanım:** AMD Ryzen 7 9700X (8 Cores, 16 Threads) (5.75 GHz OC)
- **İşletim Sistemi:** Debian Linux (WSL2)
